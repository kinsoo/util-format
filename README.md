# util-format

node格式化工具库，包含各种格式化函数

## Getting Started
Install the module with: `npm install util-format`

## Documentation
支持以下格式化函数:

+ date 日期
+ currency 货币
+ string_max 填补较长的字符串
+ mask 遮蔽敏感字符串
+ mask_mobile 遮蔽手机号
+ mask_name	 遮蔽姓名
+ mask_bankcard 遮蔽银行卡号
